Pour lancer Jupyter depuis sage,c'est simple (et on espère que ça marche aussi sur w$...) :

```shell
┌────────────────────────────────────────────────────────────────────┐
│ SageMath version 8.5, Release Date: 2018-12-22                     │
│ Using Python 2.7.15. Type "help()" for help.                       │
└────────────────────────────────────────────────────────────────────┘
sage: !sage -n jupyter
```
